1. To run the Crawler using the executable jar file

java -jar Crawler.jar -u URL -m NUMBER OF PAGES(max 50) -doc DOWNLOAD DIRECTORY -q query(optional)

The arguments can be entered in any order

eg:
java -jar Crawler.jar -u http://cs.nyu.edu/courses/spring16/CSCI-GA.2580-001/MarineMammal/Whale.html -m 10 -docs /Users/Rachit/Documents/websearch/i_crawler/output/  -q species whale whales
